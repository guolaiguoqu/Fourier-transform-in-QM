# subtleties associated with the TDSE
Does the definition of the time wavefunction require a <t|? If this is the case, does |t> exist? is |t> an eigenstate of a Hermitian operator hat(t)? Does the operator satisfy the canonical relationship with the Hamiltonian?
The last question seems to be answered by Pauli's argument, as well as Stone's theorem. However, in order to figure out a picture of the logic construction in the arguments, I started with investing the interpretation with the position wavefuntion psi(x).
